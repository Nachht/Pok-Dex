# ⚡ Pokédex

Aplicación web desarrollada con **HTML, CSS y JavaScript** que consume la **PokéAPI** para buscar información de cualquier Pokémon y gestionar una lista de favoritos utilizando **LocalStorage**.

---

## 🚀 Características

- 🔍 Búsqueda de Pokémon por nombre.
- 📦 Consumo de la PokéAPI mediante Fetch API.
- 🖼️ Visualización del nombre e imagen del Pokémon.
- ⭐ Guardado de Pokémon favoritos.
- 💾 Persistencia de favoritos mediante LocalStorage.
- 🗑️ Eliminación individual o total de favoritos.
- 📱 Diseño responsive.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- LocalStorage
- PokéAPI

## 💻 Instalación

1. Clona el repositorio.

```bash
git clone https://github.com/TU-USUARIO/Pokedex-JavaScript.git
```

2. Ingresa a la carpeta.

```bash
cd Pokedex-JavaScript
```

3. Abre el archivo `index.html` en tu navegador.

---

## 🎯 Objetivo del proyecto

Este proyecto fue desarrollado para fortalecer conocimientos en consumo de APIs REST utilizando JavaScript, manipulación del DOM, programación asíncrona con Fetch API y almacenamiento de información en el navegador mediante LocalStorage.

---

## 🌐 API utilizada

Este proyecto consume información de la **PokéAPI**, una API pública que proporciona datos sobre los Pokémon.

https://pokeapi.co/
